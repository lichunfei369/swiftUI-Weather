# swiftUI-Weather
#### ZStack VStack HStack 

- ZStack  深度布局模式  主要可以让图片在底部 
- VStack  垂直布局模式  主要是上下竖着布局 
- HStack  水平布局模式  主要是左右布局 

#### Speacer Padding 

- Speacer 让控件靠左 靠右 靠上 靠下
